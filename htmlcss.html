What is HTML?

HTML (Hyper Text Markup Language) is markup language which is used to design web pages. HTML using for developing webpages since decades. The latest version is 5.0.

jQuery is itself written in JavaScript, and comes in the form of a single .js file that you link to from your webpage. Your JavaScript code then accesses the library by calling various jQuery functions.

How to use HTML?

For HTML code no need any editor or special software. We can write html tags in a ordinary notepad and we can get the result by any browser. We should make sure that the file extension should be .html or .htm

Sample format of HTML?

<html>
    <head>     // For title, register necessary styles and scripts ... 
        <title>some name</title>  //Web page title here...
    </head>
    <body>
        //Web page content.......
    </body>
</html>

Note: HTML is not a strongly typed. Even though tag not closed HTML works fine but design will change accordingly.
