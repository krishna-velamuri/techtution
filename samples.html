Jquery is simple and easy to learn. Based on given sample you can implement more logic by combining jquery properties. We can design webforms easily and effectively using jquery. Following are the basic jquery samples.

Show or Hide an element.

To hide an element
 $("#targetId").hide(); // based on an element id
 $(".className").hide(); // based on an element class 
 $("div").hide(); // based on an element
 $("input type[text]").hide(); // based on an element type
 $("div .className").hide(); // based on an element and an element class 
 

      
To show an element - Note: use show() instead of hide()
 $("#targetId").show(); // based on an element id
 -
 -

Enable or Disable an element.

To Enable an element
 $("#targetId").attr("disabled",false);
 $(".className").hide();
 $("div").hide();
 $("input type[text]").hide();
 $("div .className").hide();
 

      
To Disable an element - Note: use disabled instead of enabled
 $("#targetId").attr("disabled",true); or  $("#targetId").attr("disabled","disabled");
 -
 -
 Negitive coding
 $("#targetId").attr("disabled",false);

Set/Get control value

 $("#txtName").val(); // To get control value based on control id.
 $("#txtName").val("Sampletext"); // To set control value based on control id.
 $("#txtName").val(''); // To clear the control value.
 $("#txtName").val($("#txtUser").val()); // To set control value with another control
