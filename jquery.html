What is jQuery?

JQuery is javascript only but it made programming easy to use functions and methods to make rich applications.These functions are very easy to learn and even a designer can learn it fast. Due to these features jQuery is very popular and in high demand among the developers. jQuery is great library for developing ajax based application.jQuery is a library that makes it quicker and easier to build JavaScript webpages and web apps. Often with jQuery you can write a single line of code to achieve what would have taken 10-20 lines of regular JavaScript code.

jQuery is itself written in JavaScript, and comes in the form of a single .js file that you link to from your webpage. Your JavaScript code then accesses the library by calling various jQuery functions.

What is document.ready()?

Jquery code runs whenever page loads. Here code will execute once document is ready, it will not wait untill download all the images on document. We can write more than one document.ready function in same web page. You can place whatever code you want execute on document is ready.

What are selectors?

Selector means selecting an element where to apply the changes. Jquery works on selectos only. We can select an element in sevaral ways.

  • Using ID.
  • Using element name.
  • Using HTML element.
  • Using class.
  • Combination above all ways.

Examples:
$("#pageContent") :"#" used to select an element based on it's ID. Here we are accessing an element which has pageContent as ID.
$("div") or $("p")..etc :To select directly a HTML element. It selects all div or p elements in the document.
$(".page") : To select an element using class name.
$("div .page") : To select a div element which has class name "page".
$("#pageContent .page") : To select an element with id (id is unique, so it selects one element from page) which has class name "page".

Syntax of jquery?

Basic syntax is: $(selector).action();

* A $ sign to define/access jQuery - $ - equalent to Jquery ( to reduce the code will use $ instead of Jquery keyword).
* A (selector) to "query (or find)" HTML elements.
* A jQuery action() to be performed on the element(s).

Examples: $(this).hide() - hides the current element.
$("div").hide() - hides all div elements in the document.
$(".header").hide() - hides all elements which are has class name as "header".
$("#content").hide()- hides the element with id="content".
